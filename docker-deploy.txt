To ssh into the running ec2 instance (keep ec2-user as-is, find ec2-instance-pub-ip-addr):
ssh -i my-ec2-key-pair.pem ec2-user@<EC2-INSTANCE-PUBLIC-IP-ADDRESS>

1) To install Docker on an AMI:
sudo yum update -y
sudo yum install -y docker
sudo service docker start
2) Add ec2-user to docker group to execute docker commands without sudo
sudo usermod -a -G docker ec2-user
exit
# ssh back in
docker info
3) Run the training/webapp, or whatever app to be run:
docker run -d -p 80:5000 training/webapp:latest python app.py

RUNNING LOCALLY
1) docker build -t <tag-name, optional> .
2) docker run --network host <id or tag-name>
3) docker stop <name or id>
